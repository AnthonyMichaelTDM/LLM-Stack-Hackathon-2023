# LLM Stack Hackathon 2023

## virtual environment

- run `python -m venv env`
- run `source env/bin/activate`

## Install dependencies

- `pip install -r requirements.txt`

## save requirements

- `pip freeze > requirements.txt`

## Tech Stack

- OpenAI api - llm
- Qdrant - vector database
- pandas/tqdm - data exploration
- react - frontend
